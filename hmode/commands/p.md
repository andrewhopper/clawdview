# Shortcut: p → /push

Execute `/push` with all provided arguments.

This shortcut provides quick access to git commit and push operations.

**Usage:**
```
/p [message] [files] [push] [dry_run]
```

**Examples:**
```
/p
/p message="feat: add auth"
/p push=false
```
