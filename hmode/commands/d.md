# Shortcut: d → /visualize

Execute `/visualize` with all provided arguments.

This shortcut provides quick access to diagram generation (mind maps, flowcharts, architecture diagrams).

**Usage:**
```
/d [file] [options]
```

**Examples:**
```
/d design.md --mindmap
/d architecture.md --architecture
/d process.md --flowchart
```
