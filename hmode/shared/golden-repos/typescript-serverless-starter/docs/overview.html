Make Commands

Setup

make install
Install all dependencies
make setup-auth-params
Setup SSM parameters

Development

make dev
Start frontend dev server
make build
Build all packages

Infrastructure

make infra-bootstrap
Bootstrap CDK
make infra-deploy
Deploy to AWS
make infra-destroy
Tear down stack

Utilities

make update-frontend-env
Update .env from CDK outputs
make outputs
Show deployment outputs
