Core Entities

CodingSession
Interactive coding environment with compute, terminals, and AI
  • id, userId, name
  • status: CodingSessionStatus
  • computeConfig: ComputeConfig
  • environmentConfig: EnvironmentConfig
  • aiAssistant: AIAssistantConfig
  • terminals: Terminal[]
  • duration, expiresAt
Terminal
PTY/shell session within a coding session
  • id, sessionId, name
  • status: TerminalStatus
  • shell, cwd
  • rows, cols (dimensions)
  • connectionProtocol: websocket|mqtt|ssh
  • connectionId

Configuration Entities

ComputeConfig
Compute resource allocation
  • type: ECS|EC2|Firecracker|Lambda
  • size: small|medium|large|gpu
  • cpu, memory, storage
  • gpu, gpuType
  • region
EnvironmentConfig
Development environment setup
  • baseImage, imageTag
  • envVars: object
  • secretRefs: string[]
  • gitRepo: GitRepoConfig
  • workspaceMount
  • packages, startupScript
GitRepoConfig
Git repository clone configuration
  • url (https or ssh)
  • branch, commit
  • path (clone destination)
  • sshKeyId
AIAssistantConfig
AI coding assistant settings
  • provider: claude-code|aider|copilot
  • model (claude-3-5-sonnet, etc.)
  • enabled: boolean
  • apiKeySecretId
  • customConfig: object

Supporting Entities

SSHKey
User SSH key for git authentication
  • id, userId, name
  • publicKey (OpenSSH format)
  • fingerprint (SHA256)
  • privateKeySecretId
  • lastUsedAt, expiresAt
APIKey
API key for programmatic access
  • id, userId, name
  • keyPrefix (first 8 chars)
  • keyHash (bcrypt)
  • permissions: string[]
  • lastUsedAt, expiresAt, revokedAt
ShareLink
Collaboration link for session sharing
  • id, sessionId, code
  • permissions: view|interact|edit|admin
  • maxUses, usedCount
  • expiresAt, revokedAt
  • createdBy
SessionUsageMetrics
Usage metrics for billing
  • sessionId, userId
  • computeMinutes, cpuUnits
  • memoryMb, storageGb
  • networkIngressMb, networkEgressMb
  • aiTokensUsed, cost
