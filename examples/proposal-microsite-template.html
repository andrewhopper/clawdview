Advanced Features

Prompt Decomposition

Automatically split complex tasks into subtasks and route each to the optimal model.

# Complex task → 4 subtasks
Extract → spaCy ($0.0001)
Sentiment → BERT ($0.001)
Analyze → Qwen ($0.30)
Summary → Haiku ($0.80)
Total: $1.18 vs $15+

Batch Queue

Queue low-priority requests for cheaper batch inference with spot pricing.

Spot pricing 60-90% off

Ollama - Easy setup

$0 cost • Data never leaves

Simple API

example.py 
from model_router import Router, Budget

router = Router()

# Simple routing - finds cheapest model for the task
response = router.route(
    prompt="Analyze sentiment of this review",
    capabilities=["sentiment"],
    quality=0.8,
)
# → Routes to sklearn-bert ($0.001/1M)

# Full constraint specification
response = router.route(
    prompt="Analyze this financial chart and extract metrics",
    capabilities=["vision", "extraction", "finance"],
    latency="<10s",
    quality=0.9,
    context=50000,
    budget=Budget(
        max_cost=0.05,  # $0.05 max
        max_tokens=10000,
    ),
)
# → Routes to GPT-4o-mini, escalates to GPT-4o if needed
