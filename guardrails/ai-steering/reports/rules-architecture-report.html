Traits (Like React Components)

Traits are composable behaviors that can be mixed into any artifact type.

📦 Publishable

Any artifact that can be published to S3

File Patterns: *.html, *.pdf, *.svg, *.zip, *.mp3, *.mp4, *.png, *.jpg

🔒 TypeSafe

Code requiring strong typing from shared domains

Languages: TypeScript (Zod), Python (Pydantic)

🎨 UIComponent

React/UI components with design system

Requires: design-system-check, storybook-story

📊 DataModel

Entities and domain models

Primitives: Artifact, Workflow, Rule, Guardrail, Validation
