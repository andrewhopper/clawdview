## Usage Recommendations

### Commander's Intent
Use when planning ANY project where:
- Team needs autonomy within boundaries
- Conditions may change mid-execution
- Clear success criteria needed
- Constraints must be communicated
- Decision authority must be delegated

### Work Back Plan
Use when planning ANY deadline-driven project where:
- Fixed deadline exists
- Dependencies between tasks
- Resource constraints matter
- Risk identification important
- Timeline must be realistic
